# 寝過ごし・乗り過ごし防止アプリ

このアプリは、降車する駅を登録しておくことで、電車などの乗り物で近づいた際に通知を送ることで、寝過ごしや乗り過ごしを防止する iOS アプリです。

## 主な機能

- 指定した駅への接近通知
- 通知距離の設定

## 使用技術

- **React Native (Expo)**
- **使用ライブラリ**:
  - react-native-maps: 地図表示と位置情報
  - axios: API 通信
